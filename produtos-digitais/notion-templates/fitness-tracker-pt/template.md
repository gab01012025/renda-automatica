Este template está desenhado para ajudar a gerir a sua jornada, quer esteja a tentar perder peso, ganhar massa muscular ou simplesmente manter um estilo de vida saudável. Siga as instruções abaixo para tirar o máximo proveito deste template.

## 📜 Página de Boas-Vindas

### Instruções de Utilização

1. **Importação do Template**: Utilize a funcionalidade "Import from markdown" no Notion para começar.
2. **Personalização**: Este template é totalmente personalizável. Sinta-se à vontade para ajustar os campos de acordo com as suas necessidades específicas.

## 🔧 Tutoriais: Como Personalizar

> **Nota**: Cada passo pode ser ajustado de acordo com as suas preferências.

- **Adicionar novos campos**: Clique em '+ Add a property' na base de dados e escolha o tipo adequado (Select, Multi-select, etc.)
- **Configurar fórmulas**: Utilize fórmulas para calcular automaticamente calorias totais, diferenças de peso, etc.
- **Criar novas views**: Utilize a opção 'Add a view' para criar novas vistas personalizadas em cada base de dados.

### Exemplo de Fórmula

Para calcular o déficit calórico diário, pode usar:
```
calorias_consumidas - calorias_gastas
```
